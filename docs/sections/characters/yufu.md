## 2. 生涯の詳細

### 2.1 出自と立花家への合流

由布惟信は豊後国速見郡湯布院城主の家系に生まれた。もともと大友氏の直臣として湯布院城を治める立場にあったが、戸次鑑連（後の立花道雪）の人柄と志に惚れ込み、自ら兄・碁晨（ごしん）に家督を譲って道雪の旗下に入った。

**城主の地位を捨てて一家臣となった**。この決断が、惟信の人生の方向を決定づけた。以後86歳で没するまで、約60年にわたり立花家に仕え続けることになる。

### 2.2 道雪の重臣として（1546年-1585年）

#### 初期の武功

- **天文15年（1546年）**: 秋月文種の謀反討伐に従軍。大友義鑑の命で筑前古処山城へ出陣。同僚の安東連善と共に**先登の戦功**を挙げ、初の感状を賜る。惟信19歳頃の初期の武功
- **弘治3年（1557年）**: 秋月文種が毛利元就の調略に応じて再度挙兵。道雪・臼杵鑑速の2万の兵で古処山城を攻略

#### 主要合戦

- **永禄5年（1562年）10月13日 -- 柳浦の戦い（大里）**
  - 豊前大里での対毛利戦。騎馬で疾駆し奮戦する姿は**敵味方を問わず驚嘆**させた
  - この戦いで**一番槍の戦功**を挙げる

- **永禄8年（1565年）5月 -- 立花山崖下の戦い**
  - 立花鑑載配下の猛将・**弥須図書助を討ち取る**
  - 味方の士気を大いに鼓舞した

- **永禄12年（1569年）5月 -- 多々良浜の戦い**
  - 対毛利軍の大規模会戦で奮戦
  - この戦いで兄・碁晨が戦死。惟信にとって家督を譲った兄を失う痛手となった
  - しかし退くことなく戦い抜いた

- **天正14年（1586年）-- 対島津戦**
  - 島津軍5万の筑前侵攻に際し、立花山城籠城や高鳥居城攻めなどで殊功を立てる
  - 高橋紹運が岩屋城で763名と共に玉砕した後も、宗茂と共に立花山城を死守

- **天正15年（1587年）9月 -- 肥後国人一揆討伐**
  - 同じく老齢であった十時惟由と共に**先鋒**に任じられる
  - 疾駆の勢いで敵を奇襲突破し、**大田黒城攻めで一番乗りの戦功**を挙げた
  - この時すでに60歳前後。老いてなお衰えぬ武勇を見せつけた

### 2.3 立花双翼 -- 「正」の将

道雪は孫子兵法の**「奇正相生」**を引用し、惟信を**「正」の将**、小野鎮幸を**「奇」の将**として両翼に据えた。二人は**「立花双翼」**と称揚された。

| | 由布惟信（正） | 小野鎮幸（奇） |
|---|---|---|
| 戦闘スタイル | 正攻法。堂々たる先鋒突撃 | 奇策。変幻自在の用兵 |
| 性格 | 剛毅。「天資英邁」 | 猛烈。「日本七柱の槍」 |
| 合戦参加数 | 65回 | 22の大戦 + 多数の小戦 |
| 戦傷数 | 65ヶ所 | 67ヶ所 |
| 感状数 | 70通 | 68通 |
| 得意 | 一番槍、先陣突破 | 奇襲、臨機応変の指揮 |

孫子曰く、**「奇・正両法を用いる者は戦って勝たないということが無い」**。道雪はこの教えを惟信と鎮幸の二人に体現させた。

### 2.4 小野鎮幸の発掘

由布惟信は**人材を見抜く慧眼**にも優れていた。小野鎮幸の才能を見出し、道雪に推薦したのは惟信だったとされる。二人の「双翼」関係は、単なる上下関係ではなく、互いの強みを認め合う**対等な戦友**としての信頼関係であった。

### 2.5 立花四天王としての位置づけ

立花四天王の構成:

| 四天王 | 特徴 |
|---|---|
| **由布惟信** | 筆頭。「正」の将。65戦65傷の武勇と政治手腕 |
| 十時連貞 | 大津城で敵将3人捕縛。宗茂への忠義 |
| 安東家忠 | 道雪七家老の四番手。武勇の将 |
| 高野大膳（諸説あり） | 四番目は内田鎮家とする説もある |

惟信は四天王の筆頭として、武勇のみならず**政治・外交面でも活躍**。天正8年（1580年）には**博多津の東分役**を任され、立花家の内政を統括する手腕を発揮した。

## 3. 道雪の遺訓を実行したエピソード

### 3.1 道雪の死と遺言（天正13年/1585年）

立花道雪は高良山の陣中で病没した。その遺言は:

> **「私が死んだなら、屍に甲冑を着せ、高良山の好己の岳に、柳川のほうに向けて埋めよ」**

この遺言は、死してなお敵（筑後の龍造寺・島津）に向かって戦い続ける意志を表したものであった。

### 3.2 殉死の危機と惟信の決断

道雪の死を受けて、家臣たちは動揺した。

- 家臣たちは**「道雪公をひとりこの地に葬れば、名も無き雑兵の馬蹄にかけられる恐れがある。我らも腹を切って供をせん」**と殉死を志した
- 由布惟信自身も切腹して付き添う意志を示した

しかし、原尻宮内少輔が**「統虎公（宗茂）のためにこそ、腹を召されよ」**と諫言。

惟信はこの言葉に応え、殉死を思いとどまり、こう宣言した:

> **「大殿のご遺骸は立花へお運びもうそう。もしご遺言のごとく祟らせ給うのであれば、わが一門にて引き受けもうす」**

道雪の遺言に背いて遺骸を運び帰ることの呪いを、**自分の一族で引き受ける**と言い切った。この覚悟をもって、惟信は道雪の遺骸を丁重に立花山城へ運び、殉死しようとする家臣たちを説得して立花家の存続を守った。

**死者への忠義と、生者への責任の間で「生きて仕え続ける道」を選んだ** -- これが由布惟信という人物の本質である。

## 4. 宗茂の教育に関わったエピソード

### 4.1 毬栗（いがぐり）事件

道雪が、まだ高橋紹運の元にいた幼い宗茂（千熊丸）を立花山城に遊びに招いた時のこと。

山道を移動中、**宗茂の足に栗のイガが刺さった**。あまりの痛みに、供の者にイガを抜くよう懇願する幼い宗茂。

しかし近寄ってきた由布惟信は、宗茂の足を掴むと**イガをさらにねじ込んだ**。

さらなる痛みに声を上げようとした宗茂だったが、目の前には「雷神」「鬼道雪」と呼ばれた立花道雪が睨みつけていたため、**声を出せなかった**。

後に立花家の養子となり、道雪の鬼の仕込みに耐えて「西国無双」の名将となった宗茂は、晩年にこの日のことを由布惟信と共に懐かしんだという。

**「痛みに耐える力」を幼少期から叩き込む** -- 由布惟信の教育は厳しいが、その先に成長がある。

### 4.2 道雪からの教育係の任命

道雪は大友家臣の高橋紹運の嫡男・千熊丸（宗茂）を観察し、将来大器になると見込んだ。そして**家老・由布惟信に宗茂の教育を託した**。

道雪が惟信を選んだ理由は明白である。65回の実戦経験から来る武芸の指導力、「天資英邁にして剛毅」な人柄、そして何より**「正」の将としての正道を歩む姿勢**が、次代の当主を育てるにふさわしいと判断したのだ。

## 5. 宗茂の浪人時代に随従し続けた詳細

### 5.1 関ヶ原の戦いと改易（慶長5年/1600年）

慶長5年（1600年）、関ヶ原の戦いで立花宗茂は西軍に与した。大津城攻略などに参加するも、本戦に間に合わず西軍は敗北。宗茂は柳川10万9,200石を改易された。

### 5.2 浪人生活への随従

10万石の大名から一夜にして浪人となった宗茂。日々の食にも事欠く困窮の生活が始まった。

多くの家臣が離散する中、**由布惟信は宗茂の側を離れなかった**。この時、惟信はすでに**73歳前後**の高齢であった。

73歳の老将が、何の保証もない浪人の主君に付き従い続けた。城主の地位を捨てて立花家に入った若き日の決断と同じく、惟信は**利ではなく義で動く人間**であった。

### 5.3 棚倉藩での再起（慶長8年/1603年）

慶長8年（1603年）、宗茂の人柄を見出した徳川家康が陸奥国棚倉藩1万石を与えた。

宗茂が江戸で将軍に近侍する間、**惟信は子の惟次と共に棚倉での藩政を担当**した。76歳を超える高齢でありながら、単なる「付き従う」だけでなく、**実務の中核を担い続けた**。

### 5.4 柳川再封後（元和6年/1620年）

元和6年（1620年）、宗茂が旧領・筑後柳川10万9,200石を回復。由布惟信（すでにこの世にはいないが）の子・惟次が宿老として各種文書や内政・外交事務を担当し、父の志を継いだ。

- **65回**: 1回も欠かさず戦場に立ち続けた「継続」
- **65ヶ所の傷**: 1回ごとに命を懸けた「全力」
- **感状70通**: 65回のうち70通の感状（複数の感状を得た戦もある）。つまり**ほぼ全ての戦で特筆すべき功績を挙げた**

「毎回出る」「毎回全力」「毎回結果を出す」-- これが由布惟信の武勇の本質であり、**Sustain（道）カテゴリの完璧な体現**である。

## 7. 性格・人柄を示すエピソード

### 7.1 「天資英邁にして剛毅」

- **英邁（えいまい）**: 才知が優れていること
- **剛毅（ごうき）**: 意志が強く、何事にも屈しないこと

惟信は単なる猛将ではなかった。武勇と知略を兼ね備え、博多津の行政を任されるほどの**政治手腕**も持ち合わせていた。

### 7.2 人材を見抜く慧眼

小野鎮幸の才能を見出して道雪に推薦し、立花双翼の「奇」として活躍させた。自分の知行を削ってでも有能な人材を迎え入れようとした逸話がある。**自分の地位や利益よりも、主家の強化を優先する**姿勢。

### 7.3 城主の地位を捨てた覚悟

湯布院城主の嫡子として生まれながら、道雪の人柄に惚れ込んで家督を兄に譲った。独立した城主から一家臣への身分変更は、戦国時代において異例の決断。**「仕えるべき主を自ら選び、その道を貫いた」**。

### 7.4 殉死を選ばなかった勇気

道雪の死に際し、殉死は武士として最も「名誉ある」選択肢であった。しかし惟信は**「生きて宗茂に仕え続ける」**という、より困難な道を選んだ。死ぬ覚悟よりも、**生きて責任を果たし続ける覚悟**のほうが重い。

### 7.5 老いてなお衰えぬ闘志

天正15年（1587年）、60歳前後で肥後国人一揆の先鋒を務め、一番乗りを果たした。73歳で浪人生活に入り、76歳で藩政を担当し、86歳まで仕え続けた。**年齢を言い訳にしなかった**。

## 8. 感状70通の意味と価値

### 8.1 感状（かんじょう）とは

感状とは、主君が家臣の戦功を称えて発行する**公式の功績証明書**。現代で言えば、社長名で発行される最高ランクの表彰状に相当する。

### 8.2 70通の異常さ

戦国時代、一生涯で数通の感状を受けることすら武士として誇りとされた。**70通**という数字は桁違いである。

しかも、これは「雷神」「鬼道雪」と恐れられた立花道雪からの感状である。道雪は容易に人を褒める人物ではなかった。70通の感状は、道雪をして「この功績は記録に残さねばならない」と認めさせた**70回の卓越した働き**を意味する。

### 8.3 アプリとの接続

由布惟信の「65回の合戦で70通の感状」は、**「毎日のチェックインで毎回フィードバックを得る」**というアプリの構造と直接対応する。

## 10. 小野鎮幸との関係 -- 立花双翼の「正」と「奇」

### 10.1 孫子の奇正相生

孫子の兵法において「奇正相生」とは:

- **正（せい）**: 定石に従って戦う本隊。正面から堂々と攻める
- **奇（き）**: 奇策によって敵の弱点を突く部隊。変幻自在

両方を使いこなすことが勝利の鉄則であり、道雪はこれを由布惟信と小野鎮幸の二人に体現させた。

### 10.2 二人の対比

| | 由布惟信（正） | 小野鎮幸（奇） |
|---|---|---|
| 年齢 | 年長（1527年生） | 年少（1546年生） |
| 性格 | 剛毅。磐石の安定感 | 猛烈。攻撃的な闘志 |
| 戦い方 | 正攻法。先陣を切って突進 | 奇策。状況に応じた臨機応変 |
| 負傷 | 両足を撃たれても地面で指揮継続（鎮幸） | -- |
| 政治力 | 博多津行政。藩政担当 | 軍事に特化 |
| 最期 | 86歳まで生き、仕え続けた | 浪人中に67歳で病没 |

### 10.3 鎮幸の才能を見出した惟信

惟信は鎮幸の武才を見抜き、道雪に推薦した。自分の地位が脅かされることを恐れず、むしろ**主家のために最善の人材配置を追求した**。

この「人を見る眼」と「自分より有能な人材を認め、推薦する度量」が、由布惟信の英邁さの一面である。

### 10.4 双翼として機能した理由

惟信の「正」と鎮幸の「奇」は互いに補完し合った:

- 惟信が正面から堂々と攻めることで、敵は惟信に対応せざるを得ない
- その隙を鎮幸が奇策で突く
- 逆に鎮幸の奇襲が混乱を起こし、惟信の正面突破が容易になる

**継続の力（正）が変化の力（奇）を支え、変化の力が継続の力を活かす** -- この関係性は、Sustain（継続）と他のカテゴリの関係にそのまま当てはまる。

## 12. 由布惟信の本質 -- なぜ「Sustain（道）」なのか

由布惟信の人生を一言で表すなら、**「止まらなかった男」**。

- 城主の地位を**止めずに**捨てた（より大きな道を選んだ）
- 65回の合戦に**一度も退かず**参加し続けた
- 道雪の死後も**殉死を選ばず**、生きて仕え続けた
- 宗茂の改易後も**離れず**随従した
- 73歳で浪人になっても**止まらず**棚倉の藩政を担った
- 86歳で没するまで**立花家への忠誠を止めなかった**

「続けること」それ自体が由布惟信の武勇であり、忠義であり、生き方であった。

アプリにおいて、由布惟信は**「チェックインを続けること自体に価値がある」**というメッセージの体現者となる。派手な成果がなくても、スコアが低くても、**記録をつけ続けること、道を歩み続けることの尊さ**を、65戦65傷の背中で語る。
