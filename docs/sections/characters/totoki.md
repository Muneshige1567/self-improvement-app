「連（つら）」の字は、主君・戸次鑑連（立花道雪）の諱の一字を拝領したもの。これは道雪から深い信頼を受けていた証拠である。

## 2. 生涯の詳細

### 2.1 出生と家督相続

- 弘治3年（1556年）、豊後国の大友氏重臣・戸次鑑連（後の立花道雪）に仕える十時惟次の次男として誕生
- 永禄12年（1569年）、**多々良浜の戦い**において父・惟次と兄・惟則が戦死
- 14歳にして家督を相続。筑前国那珂郡板付村に30町歩の所領を継ぐ
- 幼くして家を背負い、戦場で父と兄を失った経験が、後の「迷わず動く」精神の原点となった

### 2.2 宗茂との出会い -- 「ただ一人の小者」

- 立花道雪が高橋紹運の嫡男・統虎（後の宗茂）を養子に迎えた際、連貞は**「ただ一人の小者として宗茂に付けられた」**
- これは道雪が連貞を最も信頼していた証拠。養子に付ける最初の従者として選ばれるのは最高の信任
- 以後、連貞は生涯にわたって宗茂と共に歩み続ける

### 2.3 立花家臣としての昇進

| 年 | 出来事 | 知行 |
|---|---|---|
| 1569年 | 家督相続 | 30町歩 |
| 1587年 | 九州征伐で宗茂の生母・弟を救出 | 33町歩に加増 |
| 1591年 | **家老に任命** | - |
| 1592年 | 朝鮮出兵（碧蹄館の戦い参戦） | - |
| 1596年 | 朝鮮での功績 | 1,300石に加増 |
| 1598年 | 更なる加増 | 1,500石 |
| 1600年 | 関ヶ原（大津城の戦い） | 改易で全て失う |
| 1606年頃 | 宗茂の棚倉藩復帰に随従 | 200石 |
| 1620年 | 柳川復帰 → 隠居 | 隠居料1,000石 |
| 1637年 | 島原の乱に参陣（82歳） | - |
| 1644年 | 死去 | - |

### 2.4 九州征伐での武功（1587年）

- 島津軍による九州制圧で、宗茂の実母と弟が薩摩に幽閉されていた
- 天正15年（1587年）、豊臣秀吉の九州征伐に際し、連貞がこの二人を救出
- 主君の家族を救うという極めて重大な任務を任され、見事に完遂
- この功績により33町歩に加増

### 2.5 朝鮮出兵（1592-1598年）

- 天正20年（1592年）、宗茂と共に朝鮮へ渡海
- **碧蹄館の戦い（1593年）**に参戦（詳細は4.1で後述）
- 一族の十時惟道は碧蹄館で明軍の李如梅の毒矢を受け戦死
- 戦功により文禄5年（1596年）に1,300石、慶長3年（1598年）にさらに200石を加増

### 2.6 関ヶ原の戦いと改易（1600年）

- 慶長5年（1600年）、宗茂が西軍に与すると連貞も随従
- **大津城の戦い**で敵将3人を捕縛する武勇を発揮（詳細は4.2で後述）
- 西軍敗北。宗茂は改易され、10万石の大名から一夜にして浪人に
- **連貞は迷わず宗茂に従い、共に浪人となる**

### 2.7 浪人時代（1600-1606年）

- 改易後、宗茂と共に加藤清正の食客となる
- 慶長8年（1603年）冬、加藤家を辞去し宗茂と共に江戸へ
- **虚無僧となって尺八を吹き、宗茂の生活費を稼ぐ**
- 陪臣から虚無僧へ。それでも主君のそばを離れなかった
- この時期に江戸で暴漢に襲われる事件が発生（詳細は7.2で後述）

### 2.8 復帰と晩年

- 宗茂が徳川家康に見出され、陸奥棚倉藩1万石で大名復帰すると連貞も復帰
- 長年の忠義を評価され200石を拝領
- 元和6年（1620年）、宗茂の柳川復帰が許されると、連貞は老齢を理由に隠居を申し出
- 三男・十時惟昌が家督を相続
- 宗茂から隠居料として**1,000石**を与えられる（浪人時代の苦労への報いか）
- 寛永14年（1637年）、**島原の乱に82歳で参陣**。高齢の宗茂（71歳）と共に出陣
- 寛永21年（1644年）9月14日、約89歳で死去
- 福岡県柳川市の西方寺に埋葬

### 2.9 子孫

- 連貞の家系は柳川藩の組頭家兼家老家を初めとし、多くが藩士として幕末まで続いた
- 立花家への忠義は一代で終わらず、家系として受け継がれた

## 3. 立花四天王としての位置づけ

### 3.1 立花四天王とは

天正6年（1578年）の耳川合戦後、筑前の国人衆が反乱を起こした時期に活躍した4人の猛将:

| 四天王 | 特徴 | 道雪からの評価 |
|---|---|---|
| **由布惟信** | 「正」の将。65戦65傷 | 「天資英邁にして剛毅」 |
| **十時連貞** | 「沈勇にして剛直」 | 宗茂に付ける最初の従者に選出 |
| **安東家忠** | 猛将 | 「武勇絶倫的猛将」 |
| **高野大膳** | 援軍として立花軍に派遣 | 「剛勇無匹之士」 |

### 3.2 連貞の位置づけ

- 四天王の中で連貞は「沈勇にして剛直」と評される独特の存在
- 由布惟信が「華のある武勇」なら、連貞は「静かで確実な実行力」
- 派手さはないが、任された仕事を確実に遂行する。それが連貞の武勇
- 道雪が宗茂の最初の従者に連貞を選んだのは、この「確実さ」を買ったため
- **陪臣（大名の家臣の家臣）の身でありながら、その忠勇は諸大名に知られていた**

### 3.3 「陪臣なのに諸大名に知られた」意味

戦国時代において、陪臣が諸大名に名を知られることは極めて異例:

- 通常、大名と直接やり取りするのは大名同士か、せいぜい大名の家老クラス
- 連貞は立花家の家老とはいえ、立花宗茂自体が豊臣政権下の一大名に過ぎない
- その家臣が「諸大名に知られた」ということは、連貞の忠勇がそれだけ突出していたことを意味する
- 大坂の陣で豊臣方から直接高禄で誘われたのも、その名声あってこそ

## 4. 主要な戦歴・武功

### 4.1 碧蹄館の戦い（1593年）

#### 背景
- 文禄の役（朝鮮出兵）において、明の提督・李如松率いる大軍と碧蹄館で激突
- 立花宗茂は先陣として2,500の兵を率いて参戦

#### 十時一族の活躍
- 午前2時頃、森下釣雲と十時惟由ら軽兵30名が敵情を偵察
- 敵軍が未明のうちに進軍すると予測し、午前6時頃に碧蹄館南面の礪石嶺北側に布陣
- 先鋒500を率いた十時惟道と内田統続が正面で明軍と交戦
- 十時惟道、安田国継らは突撃を敢行し、鑓を投げて数十騎を突き落とす
- **十時惟道が李如梅の毒矢を受け、帰陣後まもなく戦死**

#### 連貞の立場
- 連貞は家老として軍の指揮・統率に関わった
- 一族の惟道を失うという犠牲を払いながらも、立花軍は碧蹄館で大勝
- 小早川隆景が「立花家の3,000は他家の1万に匹敵する」と評した戦いの当事者

### 4.2 大津城の戦い（1600年） -- 敵将3人捕縛

#### 背景
- 慶長5年（1600年）9月、関ヶ原の戦いの前哨戦
- 西軍の立花宗茂は近江国大津城を攻略
- 城主は東軍の京極高次

#### 連貞の武功
- 9月12日夜、宗茂は城方の**夜襲を予見**していた
- 案の定、城方が夜襲を仕掛ける
- 連貞は自らの手で敵将3名を捕縛:
  - **丸毛萬五郎**
  - **箕浦備後**
  - **三田村安右衛門**
- 一夜にして敵将3人を捕らえるという、戦国時代でも稀な武功
- この戦功は「迷わず動き、確実に仕留める」連貞の真骨頂

#### 歴史的意義
- 大津城攻略に時間を費やしたため、立花軍は関ヶ原本戦に間に合わず
- 結果的に西軍敗北。宗茂の改易につながった
- しかし連貞の個人武勇は光り続けた

### 4.3 島原の乱（1637年）

- 寛永14年（1637年）、82歳にして参陣
- 高齢の宗茂（71歳）と共に出陣
- 50年以上にわたる戦歴の最後を飾る
- 老兵が刀を置かない。その姿勢が連貞の武士道そのもの

## 5. 性格・戦闘スタイル

### 5.1 性格: 「沈勇にして剛直」

| 要素 | 解説 |
|---|---|
| **沈勇** | 静かな勇気。騒がず、動揺せず、冷静に判断し、迷わず行動する |
| **剛直** | 妥協なき直進。曲がったことを許さない。信じた道を真っ直ぐに歩む |

これは由布惟信の「天資英邁にして剛毅」とは異なる武勇の形:
- 由布惟信 = 英邁（才気煥発）で剛毅（強く屈しない） → 攻めの武勇
- **十時連貞 = 沈勇（静かに揺るがない）で剛直（真っ直ぐ折れない） → 確実な実行力**

### 5.2 戦闘スタイル

- **先読み型**: 大津城で夜襲を予見し、敵将を待ち受けて捕縛
- **確実遂行型**: 宗茂の生母・弟の救出、敵将3人の捕縛など、任務を確実にこなす
- **冷静沈着**: 虚無僧時代に3人の暴漢に襲われても、まず逃走を試み（宗茂への配慮）、追い詰められてから尺八で応戦して3人を倒す
- 派手な一番槍や一番乗りの記録は少ないが、**重要な局面で確実に結果を出す**タイプ

### 5.3 忠義の質

連貞の忠義は「盲目的な従順」ではなく「覚悟に基づく選択」:

1. **改易後も宗茂に従う** → 利害計算では離れるのが合理的。それでも残った
2. **虚無僧になってでも宗茂を支える** → 武士のプライドよりも主君への忠義を選んだ
3. **大坂の陣で豊臣方の高禄を拒絶** → 「二君に仕えず」。金で動かない
4. **82歳で島原の乱に参陣** → 老いてなお刀を置かない

## 6. 宗茂との関係性

### 6.1 関係の本質: 「最初の従者にして最後の忠臣」

- 道雪が宗茂に付けた「ただ一人の小者」として関係が始まる
- 以来、碧蹄館、大津城、関ヶ原、浪人生活、棚倉、柳川復帰、島原の乱まで、**生涯を通じて宗茂と共に歩む**
- 宗茂がまだ何者でもなかった時から、天下に名を知られた後も、全てを失った時も、復帰した時も、常に傍にいた

### 6.2 宗茂から見た連貞

- 隠居料1,000石は異例の厚遇。浪人時代の苦労と忠義への最大の敬意
- 棚倉復帰時の200石も「長年の忠義を評価して」と明記
- 宗茂にとって連貞は、単なる家臣ではなく「共に苦難を歩んだ同志」

### 6.3 連貞から見た宗茂

- 「二君に仕えず」の精神で、豊臣方の誘いを即座に拒絶
- 虚無僧に身をやつしてでも宗茂の生活費を稼いだ献身
- 82歳で島原の乱に参陣したのは「宗茂が行くなら自分も行く」の一念

## 7. 名エピソード集

### 7.1 大津城の夜襲 -- 敵将3人一夜捕縛

**状況**: 慶長5年9月12日夜、大津城攻めの最中
**経緯**:
- 宗茂が城方の夜襲を予見。防備を固める
- 夜陰に紛れて城方が出撃
- 連貞は自ら出陣し、混戦の中で丸毛萬五郎、箕浦備後、三田村安右衛門の3人を次々と捕縛

**意義**: 夜戦という視界の効かない状況で、一人で敵将3名を生け捕りにする。これは単なる武力だけでなく、冷静な判断力と行動力の証明。

### 7.2 虚無僧の尺八 -- 江戸の暴漢撃退事件

**状況**: 宗茂の浪人時代、連貞は虚無僧として江戸の町で尺八を吹き、生活費を稼いでいた
**経緯**:
1. 町外れで3人の男に襲撃される
2. 連貞はまず**逃走を選択** → 宗茂に迷惑をかけることを恐れたため
3. しかし追撃が止まない
4. 追い詰められた連貞は、**尺八で刀を受け止めて**反撃
5. 3人を返り討ちにする

**その後**:
- 奉行に捕縛される
- 事件が老中・土井利勝の耳に入る
- 土井利勝の計らいで釈放

**意義**:
- 最初に「逃げた」のが連貞の知性。武士のプライドではなく、主君への影響を優先した
- しかし追い詰められれば「迷わず戦う」。これが「沈勇」
- 尺八で刀を受け止めるという型破りな戦法も、「手段を選ばず確実に生き延びる」実行力

### 7.3 「二君に仕えず」 -- 大坂の陣での拒絶

**状況**: 慶長19年（1614年）、大坂の陣を前に豊臣方が各地の武将を勧誘
**経緯**:
- 豊臣秀頼から高禄をもって仕官を招かれる
- **陪臣の身**でありながら豊臣方に名を知られていたこと自体が異例
- 連貞は即座に拒絶

**意義**:
- 関ヶ原で西軍として戦った連貞にとって、豊臣方への恩義がないわけではない
- それでも「今の主は宗茂ただ一人」と貫いた
- この拒絶こそが「二君に仕えず」の体現

### 7.4 宗茂の母と弟の救出

**状況**: 天正15年（1587年）、島津軍による九州制圧時
**経緯**:
- 宗茂の実母と弟が薩摩に幽閉されていた
- 豊臣秀吉の九州征伐に際し、連貞が救出任務を担う
- 無事に二人を救出して帰還

**意義**: 主君の家族を救うという最も重要な任務を任され、確実に遂行。道雪が連貞を宗茂に付けた判断の正しさを証明。

### 7.5 82歳の島原出陣

**状況**: 寛永14年（1637年）、島原の乱
**経緯**:
- 82歳という高齢にもかかわらず参陣
- 71歳の宗茂と共に出陣

**意義**: 50年以上にわたって宗茂と共に戦い続けた。「老兵は刀を置かない」を体現。

## 8. アプリキャラクターとしての設計案

### 8.1 役割: Execute（剣）カテゴリ担当

**「迷わず実行する」象徴**

連貞は「沈勇にして剛直」。静かに、しかし迷いなく実行する。その姿はアプリの Execute カテゴリ（実行力・決断・行動の勇気）と完璧に一致する。

### 8.2 キャラクターの核心

| 項目 | 設定 |
|---|---|
| キーワード | 「迷わず動け」「確実に仕留めろ」「手を止めるな」 |
| 精神 | 沈勇（静かな勇気）と剛直（真っ直ぐ折れない） |
| 武士道の徳 | **勇**（行動する勇気） |
| 行動原理 | 考えてから動くのではなく、動きながら考える。ただし冷静さは失わない |

### 8.3 口調設計

| 項目 | 設定 |
|---|---|
| 一人称 | 「某」（それがし） |
| 二人称 | 「お主」 |
| 語尾 | 「〜だ」「〜ぞ」「〜せよ」「〜であった」 |
| 特徴 | 宗茂より言葉が少なく、実務的。無駄な装飾がない。行動で語る男の言葉 |
| トーン | 静かだが確信に満ちている。叱る時も声を荒げず、事実を突きつける |

### 8.4 フィードバック例

#### praise（称賛）-- Execute が高い時
```
「今日の"剣"、迷いがなかった。
 大津城の夜、某が敵将三人を捕らえた時と同じだ。
 考える前に身体が動く。それが極意だ。」
```

#### rebuke（叱咤）-- Execute が低い時
```
「手が止まっておるぞ。
 某は虚無僧に身をやつしてでも主のために動いた。
 できぬと嘆く暇があれば、まず一歩踏み出せ。」
```

#### encourage（激励）-- 行動を促す時
```
「完璧でなくともよい。
 刀は振らねば錆びる。
 今日の一振りが、明日の切れ味を変える。」
```

#### advise（助言）-- Execute カテゴリへの指導
```
「"剣"の極意は三つ。
 一、迷わぬこと。二、止まらぬこと。三、確実に仕留めること。
 どれが欠けておる？ 自ら考えよ。」
```

#### warn（警告）-- Execute が急落した時
```
「刀を鞘に収めたか。
 某は八十二の齢まで戦場に立った。
 手を止める理由など、この世にはない。」
```

### 8.5 連貞固有のフィードバック素材

| シナリオ | 連貞の言葉の方向性 |
|---|---|
| E1（80点を最速で） | 「完璧な一太刀を待つな。某は夜闘の中で三人を仕留めた。暗闇でも振れ。」 |
| E2（期待値を超える） | 「某は虚無僧の身で暴漢三人を退けた。立場がどうあれ、やるべきを為せ。」 |
| E3（自分から手を挙げる） | 「道雪様が某を宗茂様の傍に付けた。名乗りを上げた者にこそ大任が回る。」 |
| E4（障害をHowで突破） | 「尺八で刀を受けたこともある。手段を選ぶな。突破せよ。」 |
| E5（仕組み化・効率化） | 「同じ敵に二度苦戦するは恥。一度の戦から型を学べ。」 |

### 8.6 登場タイミング

| タイミング | 演出 |
|---|---|
| 解放条件 | ストリーク100日で解放（家臣団全員解放の一人として） |
| 初登場セリフ | 「某は十時連貞。宗茂様の最初の従者にして最後の忠臣。お主の"剣"を鍛えてやろう。」 |
| Execute 高スコア時 | 静かな肯定。「見事。迷いのない一太刀だ。」 |
| Execute 低スコア時 | 事実を突きつける。「刀が錆びておるぞ。振らねば切れぬ。」 |
| 全カテゴリ好調時 | 「五つの領域、全て隙がない。大津城の夜を思い出す。あの夜も、全てが噛み合った。」 |

### 8.7 他キャラクターとの差別化

| キャラクター | 実行力の語り方 | トーン |
|---|---|---|
| **宗茂** | 大将としての決断。碧蹄館の突撃。義に基づく選択 | 温かく、包容力がある |
| **紹運** | 死を覚悟した行動。岩屋城の最後の一振り | 熱い。覚悟を問う |
| **十時連貞** | **静かで確実な実行。夜闘での捕縛。尺八での反撃** | **冷静。事実を語る** |

連貞は「熱血型の実行力」ではなく「冷静確実型の実行力」。これは社会人としてより実践的。
