## 2. 生涯

### 2.1 出自と父の謀殺

米多比鎮久は、大友宗麟の家臣・米多比直知の子として筑前糟屋郡に生まれた。米多比氏は筑前の米多比山に居城する一族で、本姓を丹治とする。

永禄7年（1564年）、立花鑑載が大友宗麟に対して謀叛を起こした。鑑載の与力であった父・直知は大友への忠誠を保ち、挙兵に加担しなかったために**鑑載に謀殺された**。鎮久はまだ13歳の少年であった。

父を主君の裏切りによって失うという壮絶な体験が、後の鎮久の「義を重んじ、人との絆を何より大切にする」人格の原点となった。

### 2.2 薦野増時との盟友関係と初陣

永禄11年（1568年）、立花鑑載が再び大友氏に叛旗を翻すと、鑑載は薦野・米多比一族の討伐のために安武民部・藤木和泉守ら800の兵を差し向けた。

これに対し、鎮久は盟友・薦野増時と共にわずか**300の兵で迎え討ち、西郷原で800の敵を撃退する**大勝利を収めた。この戦いが鎮久と増時の武名を轟かせる最初の機会となった。

増時と鎮久はほぼ同時期に立花家に仕官し、以後生涯にわたって盟友関係を貫いた。二人は常に対で語られ、古賀市の郷土史料でも並んで紹介されている。

### 2.3 立花道雪への仕官と重用

大友宗麟の命で戸次鑑連（後の立花道雪）が立花山城城督として赴任すると、鎮久は薦野増時と共に召し出され重用された。

道雪のもとで鎮久は「立花双翼の副将」として名を馳せる。立花双翼の構成は以下の通り:

| 役割 | 武将 |
|---|---|
| 正軍の将（正面攻撃の要） | 由布惟信 |
| 奇軍の将（奇策・側面攻撃の要） | 小野鎮幸 |
| 正軍の副将 | 薦野増時 |
| 奇軍の副将 | **米多比鎮久** |

鎮久は生の松原合戦や潤野原合戦などの戦いで敵将を討ち取り、着実に武功を積み重ねた。

### 2.4 道雪からの養女 -- 人柄を愛された男

**道雪は鎮久の人となりを深く愛し、自らの養女・吉子を妻として与えた。**

吉子は道雪の後妻の連れ子で、道雪の養女となっていた人物であり、誾千代の義姉にあたる。つまり鎮久は道雪の「義理の娘婿」となったのである。

さらに道雪は鎮久に**立花姓を与え**、「立花三左衛門鎮久」と名乗ることを許した。家臣に自らの姓を許すのは最大級の信頼の証であり、道雪がいかに鎮久の人格を高く評価していたかを示す。

鎮久と吉子の間には三男三女が生まれ、家庭的にも恵まれた。この「人に愛され、人を大切にする」性質こそ、鎮久の本質であった。

### 2.5 九州平定と鷹尾城城代

天正15年（1587年）、豊臣秀吉の九州平定の功績により、宗茂が筑後国柳川城に移封されると、鎮久は**鷹尾城城代家老・3,500石**を拝領した。立花六城主の一人として、柳川藩の防衛の要を担う重責を任されたのである。

肥後国人一揆の鎮圧にも出陣し、小野鎮幸や統増と共に戦線を支え、戦功を挙げた。

### 2.6 碧蹄館の戦い（1593年） -- 虎丹波の異名

文禄の役（1592-1593年）では宗茂と共に朝鮮に出陣。鎮久は中陣の一翼を担い、小野鎮幸と共に布陣した。

文禄2年（1593年）1月26日の碧蹄館の戦いにおいて、鎮久は**数十名の敵を斬り伏せる獅子奮迅の活躍**を見せた。あまりの激闘に、**旗指物（旗印）が敵の血で真紅に染まった**。この旗指物は後世まで子孫の家に保管され、鎮久の武勇の証として伝えられた。

#### 虎退治のエピソード

朝鮮在陣中、鎮久は山に入り鉄砲で子虎を仕留めた。宗茂から「親虎が必ず報復に来る、用心せよ」と忠告を受けたところ、案の定、後日親虎が襲来。鎮久は再び鉄砲で親虎をも見事に仕留めた。

この時に使用した銃は「大虎」「小虎」と命名され、虎の歯と共に家中に保管された。この逸話から**「虎丹波（とらたんば）」**の異名で呼ばれるようになった。

### 2.7 関ヶ原の戦いと改易後 -- 誾千代を守った忠義

慶長5年（1600年）、関ヶ原の戦いで西軍についた宗茂が改易されると、立花家臣団はそれぞれの道を歩むことになった。

鎮久は**加藤清正の熊本藩に3,000石で仕官**した。しかし注目すべきは、鎮久がこの時に果たした役割である。

**鎮久は宗茂と別居中の正室・誾千代姫と、その母・宝樹院を引き取り、長きにわたって孝養を尽くした。**

宗茂が離縁した妻と義母を預けたのが鎮久であったという事実は、宗茂が鎮久を「最も信頼できる人物」と見なしていたことの何よりの証拠である。道雪の娘（義理の娘）と妻を、道雪の義理の娘婿である鎮久に託したのは、人間関係の縁を何よりも大切にする立花家の精神の発露であった。

### 2.8 牛方馬方騒動と宗茂への帰還

加藤忠広（清正の子）の時代に加藤家で起きた「牛方馬方騒動」というお家騒動に鎮久は巻き込まれ、証人として江戸に登った。その後、当時陸奥棚倉にいた宗茂の**預かり**となった。

これにより鎮久は再び宗茂のもとに身を寄せることとなった。

### 2.9 旧領回復と番頭 -- 最後の奉公

元和6年（1620年）、宗茂が大坂の陣の功績により旧領・筑後柳川10万9,200石を幕府から与えられると、**鎮久は呼び戻され、番頭として1,000石を賜った**。

番頭は藩の軍事・行政の要職であり、70歳を超えた鎮久が召し戻されたことは、宗茂が鎮久の経験と人望をいかに頼りにしていたかを物語る。鎮久は最晩年まで現役の番頭として藩政を支え続けた。

### 2.10 死去 -- 83年の忠勤

寛永10年（1633年）9月11日、米多比鎮久は83歳で死去した。法名は「空源院殿西譽榮鎮大居士」。福岡県柳川市の良清寺に葬られた。

13歳で父を失い、少年期から戦場に立ち、道雪・宗茂の二代に仕え、朝鮮で虎を退治し、碧蹄館で旗指物を血に染め、改易後は誾千代を守り、70を過ぎてなお番頭として呼び戻された。**83年の生涯は、まさに「人徳」と「忠勤」の二文字に集約される。**

## 3. 人物像・性格

### 3.1 公式評価

> **「人となり剛勇果断にして苟も人後に落ちることを恥じる。戦陣に臨み殊勲を立てること数回あり常に驍勇絶倫。」**

（性格は剛勇で果断、決して人の後ろに落ちることを恥とした。戦場に臨んでは殊勲を立てること数回、常に驍勇絶倫であった。）

### 3.2 性格の特徴

| 特徴 | 詳細 |
|---|---|
| **剛勇果断** | 碧蹄館で数十名を斬り、親虎をも仕留める武勇。しかしそれを誇らない |
| **謙虚** | **生涯、自らの戦功を人に説くことがなかった**。これは極めて稀な美徳 |
| **人徳** | 道雪に「人となりを深く愛され」養女を妻に与えられた。人間的な魅力の証 |
| **信頼される人柄** | 宗茂が最も大切な存在（誾千代と義母）を預けた相手。究極の信頼 |
| **文武両道** | 武勇だけでなく**能筆家（書道の達人）**としても称えられた |
| **忠義** | 83歳まで仕え続け、70を過ぎてなお番頭として呼び戻される忠勤 |
| **まっすぐな善意** | 「善意に満ちた」人物と評され、その誠実さが周囲からの信頼を集めた |

### 3.3 能筆家としての顔

鎮久は武人でありながら**書道の達人**としても知られた。その墨色について:

> **「美婦の傾国の色に誇るが如し」**

（傾国の美女が誇る美しさのようだ）

と讃えられたほどである。武骨な武将のイメージとは異なり、文化的な素養も深かった。これは鎮久の人間としての奥行きを示す重要なエピソードであり、「人の心を動かす力」は武勇だけでなく文化的な感性にも根ざしていたことがわかる。

## 4. 他の家臣との関係

### 4.1 薦野増時（こもの ますとき）-- 生涯の盟友

鎮久と増時は「立花双翼の副将」として常に対で語られる盟友関係にあった。

| 比較 | 薦野増時 | 米多比鎮久 |
|---|---|---|
| 性質 | 冷静・分析・先見 | 剛勇・人徳・誠実 |
| 双翼での役割 | 正軍の副将 | 奇軍の副将 |
| 関ヶ原後 | 黒田家に仕官、後に道雪の墓守 | 加藤家に仕官、誾千代を守護 |
| 共通点 | 父を鑑載に謀殺された。共に300で800を破る。道雪に重用された |

二人は少年期に共に父を失い、共に300の兵で800の敵を破り、共に道雪に見出された。まさに「戦場の盟友」であった。

### 4.2 立花道雪 -- 養女を与えた主君

道雪は数多くの家臣の中から鎮久の人柄を特に愛し、養女・吉子を妻に与え、さらに立花姓を許した。これは他の四天王にも見られない特別な待遇であり、道雪が鎮久に見出した「人としての器」の大きさを物語る。

### 4.3 立花宗茂 -- 誾千代を託した主君

宗茂が改易後、最も大切な存在を預けた相手が鎮久であった。70歳を超えてなお番頭として呼び戻したことからも、宗茂にとって鎮久が「替えの利かない存在」であったことは明白である。

### 4.4 由布惟信・小野鎮幸 -- 立花双翼の主将

由布惟信（正軍の将）と小野鎮幸（奇軍の将）が立花双翼の主将を務め、鎮久は小野鎮幸の下で奇軍の副将を担当した。主将たちが戦場の「剣」ならば、鎮久は陣中の「人の和」を支える存在であった。

### 4.5 十時連貞 -- 立花四天王の同僚

十時連貞と共に「立花四天王」に数えられる。十時が「実行力の象徴」であるのに対し、鎮久は「人徳の象徴」として対比される。

## 5. エピソードまとめ

### 5.1 「300で800を破る」（西郷原の戦い）

永禄11年、薦野増時と共にわずか300の兵で800の敵を撃退。数ではなく結束力で勝った。

**アプリへの接続**: 「人」の力は数ではなく質。信頼で結ばれた少数は、烏合の多数に勝る。

### 5.2 「道雪に養女を与えられる」

道雪は多くの家臣の中から鎮久の人柄を最も愛し、養女を妻に与え、立花姓を許した。

**アプリへの接続**: 人徳は最大の武器。能力だけでなく人柄が、最も深い信頼を勝ち取る。

### 5.3 「碧蹄館の血旗」

碧蹄館の戦いで数十名を斬り、旗指物が敵の血で染まった。その旗は子孫に代々伝えられた。

**アプリへの接続**: 本気で戦った証は残る。日々の全力の記録が、未来の自分への誇りになる。

### 5.4 「虎丹波」（虎退治）

朝鮮で子虎を仕留めた後、宗茂の忠告通り親虎が来襲。再び鉄砲で仕留め、「虎丹波」の異名を得た。

**アプリへの接続**: 仲間の助言を聞く耳を持つ。そして助言を活かして行動する。それが信頼関係。

### 5.5 「誾千代を守る」

改易後、宗茂の正室・誾千代と義母を引き取り、長く孝養を尽くした。

**アプリへの接続**: 本当の信頼は、順境ではなく逆境で試される。困った時こそ「人」の力が問われる。

### 5.6 「戦功を語らぬ武士」

生涯、自らの戦功を人に説くことがなかった。

**アプリへの接続**: 実力は見せつけるものではなく、行動で示すもの。謙虚さが人望を生む。

### 5.7 「70歳を超えて呼び戻される」

旧領回復後、70歳を超えた鎮久を宗茂は番頭として呼び戻した。

**アプリへの接続**: 長年の信頼と経験は、決して色褪せない。継続こそ最大の資産。

### 6.2 口調設計

鎮久は「剛勇果断」でありながら「戦功を語らない」人物。口調は穏やかで温かいが、芯の強さが時折覗く。

**宗茂との違い**: 宗茂は「師匠」として全体を見渡すが、鎮久は「人間関係の達人」として具体的な対人スキルを語る。

**他の家臣との違い**: 由布の「不退転の継続」、十時の「即座の行動」、小野の「不屈の闘志」、薦野の「冷静な分析」に対し、鎮久は「人との絆」に特化。

| **ストリーク100日で解放** | 初登場: 「百日か。某は八十三の歳まで歩み続けた。お主にもその素質があるようだ。」 |
| **ストリーク365日（全員祝福）** | 「一年。長い道であったな。某が道雪様に立花の姓を賜った日を思い出す。お主もまた、この道の名に恥じぬ者となった。」 |

### 6.6 character-style-guide.md との接続

| 既存設計 | 鎮久での具体化 |
|---|---|
| Engage → 「人」（じん）| 鎮久は「人徳」そのもの。道雪に人柄を愛され、宗茂に誾千代を託された |
| 仁・礼（思いやりと敬意）| 改易後も誾千代と義母に孝養を尽くす「仁」。戦功を語らない「礼」 |
| 「信頼は一日では築けぬ。だが一日で崩れる」| 鎮久の83年の忠勤がこの格言の生きた証明 |
| 対人メタファー（G1-G5）| 全て鎮久の実体験で裏打ち可能 |
